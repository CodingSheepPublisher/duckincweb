Duck Incorporated SCP:SL

Basic Info:

Duck Incorporated is a heavily modded SCP:SL server with many things like custom roles, custom items, custom map elements, and much more.
The server also has some other stand out features, such as, infinite stamina, and no item, or ammo limits.
The staff team on Duck Incorporated is also very fun, doing events fairly often.

Detailed Plugin List:

  • Uncomplicated Custom Items: Adds several custom items that are specific to the Duck Incorporated server to the game.
  • Uncomplicated Custom Roles: Adds several custom roles that are specific to the Duck Incorporated server to the game.
  • Auto Broadcast: Automates server broadcasts.
  • Advanced MER Tools: Allows for more things such as replacing rooms with MER schematics.
  • Map Editor Reborn: Allows the creation of custom maps elements.
  • CedMod V3: Allows for syncing Discord roles with server roles, and much more.
  • Micro Vaporate: Vaporizes players when they are killed by the Micro-HID.
  • Remote Keycard: Allows players to open keycard protected doors without holding their keycard out.
  • Horror Sounds: Plays spooky sounds sometimes.
  • Round MVP: Displays round statistics such as who got the most kills at the end of the round.
  • SCP-1162: Adds The Hole in the Wall to SCP-173's containment chamber.
  • SCP-3114 Spawner: Allows SCP-3114 to spawn in-game.
  • SCP Proximity Chat: Allows SCPs to use proximity chat.
  • SCP:SL Audio API: Developer functions.
  • SCP Swap: Allows SCPs to swap roles with each other.
  • Scripted Events: Automates server functions.
  • Show Reports in Game: Shows a staff member when a player was reported in game.
  • Ultimate AFK: Replaces AFK players with spectators.
  • VIP Broadcast: Allows certain players (mainly players with Patreon roles) to broadcast a message.
